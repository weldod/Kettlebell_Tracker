# Kettlebell Tracker & Timer

A simple web-based timer and session tracker built specifically for kettlebell workouts. 

[**Live Demo**](https://weldod.github.io/Kettlebell_Tracker/)

## Features

- **Customizable Intervals**: Set your work duration, rest duration, kettlebell weight, and total sets to match your workout perfectly.
- **Exercise List Management**: Add, organize, and save your custom exercise sequences.
- **Session History**: Automatically saves your past workouts for easy tracking and review.
- **PWA Ready**: Installable as a Progressive Web App (PWA) on mobile and desktop for offline use.
- **Audio Cues**: Optional sound notifications to guide you through your work and rest phases without looking at the screen.
- **"Skip Last Rest" Option**: Optimize your time by automatically skipping the rest period after your final set.

## Technologies Used

- **HTML5 & Vanilla JavaScript**: Core structure, behavior, and timer logic.
- **Tailwind CSS**: Utility-first CSS framework for fast, custom, and responsive styling.
- **Service Workers**: Enables offline functionality and PWA installation.

## Getting Started

To run the project locally:

1. Clone this repository:
   ```bash
   git clone https://github.com/weldod/Kettlebell_Tracker.git
   ```
2. Navigate to the project folder and open `index.html` in your web browser.
3. For the best experience and to test PWA features locally, serve the directory using a local development server (like VS Code Live Server or Python's `http.server`).

## Usage Info

1. **Configure Workout**: Set your exercises, specify work/rest times, kettlebell weight, and total sets per exercise in the configuration menu.
2. **Save Presets**: Quickly recall your routines by using the "Save" button to retain your current exercise list.
3. **Start Timer**: Hit the massive "Start" button to begin the timer. The main screen will guide you through your session.
4. **Review History**: View your previous workouts anytime by clicking the history icon located in the header.
